Comment joué ?

Installer Visual Studio Code
Installer l'extension Live Server
Se rendre sur le code
F1 -> Open with live server




Les déplacements de base du personnage autre que les déplacements latéraux et le saut sont:
 -un Wall Jump sur n'importe quelles surfaces
 -un Sprint augmentant la vitesse du personnage sur l'axe X, utilisable avec la touche SHIFT ou A

Les ennemies se déplace de haut en bas, pour les faires disparaitres il suffit de les touché par le haut, une collision dans un autre sens entraine la perte de PV

Pour les Power Up à débloqué, le premier est un Double Jump, le second est un Dash (Espace ou B)

L'altération de déplacement est effecuté par des block infecté par une substance étrange ralentissant le joueur
